Rapheal Suber Portfolio

This is the personal portfolio website for Rapheal Suber, a UX Designer and Developer. Ideally viewed on desktop but fully responsive for mobile devices.

Technological Stack

HTML5
CSS3 (Custom Properties for theming)
JavaScript (ES6+)
Git for Version Control
Hosted on Railway via GitHub

Features

Custom "Book Layout" Design System
Interactive Mouse Trail Animation
Bookmark Navigation Menu
Responsive Case Study Layouts
Dark Mode Interfaces (in Case Studies)

Local Development

1. Clone the repository
2. Run a local server (e.g., using http-server or Live Server)
3. Open localhost in your browser
